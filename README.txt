Этот сервис позволяет отслеживать статьи, которые вышли сегодня на сайтах N+1 и "Наука и жизнь" 
Чтобы запустить сервис, необходимо прописать следующие команды:
git clone https://github.com/surkinasasha/CodeReview.git
cd CodeReview
sh ./build.sh

Сайт работает по ссылке http://127.0.0.1:5000 
Необходимо будет подождать некоторое время, прежде чем сайт заработает
